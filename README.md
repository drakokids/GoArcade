# GoArcade
A trial to build a game engine in Go
# MainGoals
-2D Engine
-Fake 3D Shadows and Lights
-Scrollable Maps (tilemap and vector maps)
-Sprite Engine
-Particle System
-3D objects in 2D Scenes
-3D Sounds
-Gui Image Based
-International Dictionary 
-2D Physics
-Mouse, Touch, Gamepad and arcade inputs
